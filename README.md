## 🌟 프로젝트 개요

해당 repo는 **웹디자인개발기능사 실기 시험**에서 다뤄지는 유형을 연습하고,
실무 포트폴리오용 웹 페이지 예제들을 관리하기 위해 만들어졌습니다.
- 90% 정도 본 시험과 동일하게 작성
- 본 시험은 수험자 제공파일이 별도로 존재하여 이미지 파일은 없으며, 
  대체로 작성하였습니다.
- 향후 국가기술자격증 참고 사항, 큐넷 제공 공식 유형 알집 파일 첨부 예정
- 디렉토리 구조 및 기능 설명 추가 예정


- **시험**: F-3 유형
- **주제**: 오픈뱅킹 웹 사이트 제작
